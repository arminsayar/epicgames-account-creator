# Script for creating new epic games accounts | "Free accounts"

## Install

```bash
npm install
```

or

```bash
yarn add
```

## Dependencies

 [Download](https://chromedriver.chromium.org/downloads) chromdriver and add to your varible env "path" 
Install [Nodejs](https://nodejs.org/en/download/)

## Run

```bash
npm start
```
